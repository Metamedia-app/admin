**AdminPanel** adalah dashboard administrasi modern dan responsif yang dirancang khusus untuk mengelola ekosistem media sosial. Proyek ini dibangun dengan fokus pada performa tinggi, keamanan moderasi, dan pengalaman pengguna (*UX*) yang premium.

---

## ✨ Fitur Utama

### 📊 Social Media Analytics
Dashboard bergaya premium dengan grafik pertumbuhan interaksi real-time, statistik pengguna, interaksi (likes), postingan baru, dan konten yang sedang tren.

### 🛡️ Moderasi Konten & User
*   **Post Takedown:** Fitur moderasi postingan yang melanggar kebijakan.
*   **User Ban/Unban:** Kontrol penuh atas akses pengguna ke platform.
*   **Reports System:** Manajemen laporan masuk dengan status *Pending*, *Ignored*, atau *Resolved*.

### 📚 Manajemen Akademik (Grup Matkul)
*   **Subject Management:** Kelola daftar Mata Kuliah yang tersedia.
*   **Group Chat Admin:** Pantau grup chat aktif.
*   **Smart Member Selection:** Fitur tambah mahasiswa ke grup dengan *Live Search* dan validasi profil (NIM/Nama/Foto) untuk mencegah kesalahan input.

### 🎨 Design Premium
*   **Nexus Dark Sidebar:** Navigasi gelap yang elegan dengan kontras tinggi.
*   **Modern Glassmorphism:** Elemen visual yang transparan dan halus.
*   **Interactive Charts:** Grafik batang dengan gradasi dan tooltip interaktif.

---

## 🚀 Teknologi yang Digunakan

*   **Frontend Framework:** React 18
*   **Build Tool:** Vite
*   **Styling:** Tailwind CSS (dengan kustomisasi premium)
*   **Icons:** Lucide React
*   **Routing:** React Router DOM
*   **State Management:** React Context (Auth & Toast)
*   **API Client:** Axios / Fetch dengan interceptor keamanan

## 🛠️ Cara Instalasi

1. **Clone Repositori**
   ```bash
   git clone https://github.com/QUHu-FER/admin.git
   cd admin
   ```

2. **Instal Dependensi**
   ```bash
   npm install
   ```

3. **Konfigurasi Environment**
   Buat file `.env` di root direktori:
   ```env
   VITE_API_BASE_URL=https://besosmed-production.up.railway.app
   ```

4. **Jalankan Aplikasi**
   ```bash
   npm run dev
   ```
